# Digital Frontier Portfolio

Welcome to my portfolio! This repository contains the source code for my personal portfolio website, designed to showcase my projects, skills, and experience.

## Overview

A modern, high-performance portfolio built with a React frontend and a Node.js/Express backend. The application features an immersive 3D/neon aesthetic, interactive project cards, and hardware-accelerated animations to deliver a premium user experience.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Three.js / React Three Fiber
- **Backend**: Node.js, Express, Nodemailer

## Projects Featured

1. **SecureScan Pro**: An automated web application vulnerability scanner and threat intelligence platform.
2. **Narrative Flow**: A full-stack AI co-writing platform built with FastAPI, Next.js, and local AI pipelines.
3. **TribalBridge**: A cross-platform infrastructure layer for connecting decentralized communities.
